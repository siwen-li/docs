# 快速开始

## 安装和创建项目

```
# 1. 创建虚拟环境
python3 -m venv .venv

# 2. 激活虚拟环境
source .venv/bin/activate

# 3. 安装 Zensical
pip install zensical

zensical new .
```


## 构建网站 
```
# 构建网站
zensical build

# 或使用清理构建（推荐，清除旧文件）
zensical build --clean
```

## 本地预览 
```
# 使用 Python 内置服务器预览
cd site
python3 -m http.server 8001
```

## 常用命令 
```
# 启动开发服务器
zensical serve

# 启动开发服务器（指定端口）
zensical serve --port 8001

# 构建静态网站
zensical build

# 清理构建（清除旧文件）
zensical build --clean

# 查看帮助
zensical --help

# 查看版本
zensical --version
```
