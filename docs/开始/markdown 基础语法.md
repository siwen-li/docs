# Markdown 基础语法

## 1. 标题 (Headings)

Markdown 使用井号 `#` 来标记标题，井号的数量代表标题的级别（1-6级）。

**语法：**
```markdown
# 一级标题 (H1)
## 二级标题 (H2)
### 三级标题 (H3)
#### 四级标题 (H4)
##### 五级标题 (H5)
###### 六级标题 (H6)
```

**注意：** `#` 和标题文字之间最好保留一个**空格**。


---

## 2. 强调语法 (Emphasis)

你可以使用星号 `*` 或下划线 `_` 来标记粗体或斜体。

| 样式 | 语法 | 效果 |
| :--- | :--- | :--- |
| **粗体** | `**粗体文本**` 或 `__粗体文本__` | **粗体文本** |
| *斜体* | `*斜体文本*` 或 `_斜体文本_` | *斜体文本* |
| ***粗斜体*** | `***粗斜体文本***` | ***粗斜体文本*** |
| ~~删除线~~ | `~~删除文本~~` | ~~删除文本~~ |

---

## 3. 引用 (Blockquotes)

使用大于号 `>` 来表示引用块。

**语法：**
```markdown
> 这是一个引用段落。
>
> > 这是一个嵌套引用。
```

**效果：**
> 这是一个引用段落。
>
> > 这是一个嵌套引用。

---

## 4. 列表 (Lists)

### 无序列表
使用 `*`、`+` 或 `-` 作为列表标记。

**语法：**
```markdown
- 项目 1
- 项目 2
  - 子项目 A (前面加Tab缩进)
  - 子项目 B
```

**效果：**

* 项目 1
* 项目 2
    * 子项目 A
    * 子项目 B

### 有序列表
使用数字加英文句点 `.`。

**语法：**
```markdown
1. 第一步
2. 第二步
3. 第三步
```

**注意：** 列表标记符号（如 `-` 或 `1.`）后面必须跟一个**空格**。

---

## 5. 代码 (Code)

### 行内代码
使用反引号 \` 包裹代码内容。

**语法：**
```markdown
请使用 `console.log()` 打印结果。
```

**效果：**
请使用 `console.log()` 打印结果。

### 代码块
使用三个反引号 \`\`\` 包裹代码块，由于支持语法高亮，可以在开头的反引号后指定语言（如 python, javascript, java 等）。

**语法：**
````markdown
```python
def hello():
    print("Hello, Markdown!")
```
````

**效果：**
```python
def hello():
    print("Hello, Markdown!")
```

---

## 6. 链接 (Links)

链接的语法是：`[链接显示文本](链接地址 "可选的标题")`。

**语法：**
```markdown
这是一个去往 [Markdown中文网](https://markdown.com.cn) 的链接。
```

**效果：**
这是一个去往 [Markdown中文网](https://markdown.com.cn) 的链接。

---

## 7. 图片 (Images)

图片的语法与链接非常相似，只是前面多了一个感叹号 `!`。
语法：`![图片描述(Alt Text)](图片链接 "可选标题")`

**语法：**
```markdown
![Markdown Logo](https://markdown.com.cn/hero.png)
```

**注意：** Markdown 无法直接调整图片大小，如果需要调整大小，通常需要使用 HTML 的 `<img>` 标签。

## 9. 表格 (Tables)

虽然属于扩展语法，但被广泛支持。使用 `|` 分隔单元格，使用 `-` 分隔表头。

**语法：**
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :----: | -----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :----: | -----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

---

## 10. 转义字符 (Escaping)

如果你想显示 Markdown 中的特殊符号（如 `*`, `#`, `[` 等）原本的样子，可以在它们前面加上反斜杠 `\`。

**语法：**
```markdown
\*这不是斜体\*
```

**效果：**
*这不是斜体*
