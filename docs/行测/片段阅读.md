# 片段阅读

## 01. 片段阅读

### 题型分类
1.  **中心理解题：** “这段文字意在说明/意在强调/主要介绍了/核心观点是/概括最恰当的是…”
2.  **标题拟定题：** “最适合做这段文字标题的是”
3.  **下文推断题：** “这段文字接下来最可能讲的是”
4.  **语句填入题：** “填入画横线部分最恰当的一项是”
5.  **语句排序题：** “将以上6个句子重新排列，排序正确的是”
6.  **细节判断题：** “下列选项中，与这段文字意思不相符/理解错误/与文意不符的是”

### 文段分类
*   **说理类：** 表达作者的观点、看法、态度等。
    1.  原文有对策优先选对策（极少数会改选对策句）。
    2.  原文有作者观点或态度选观点态度。
    3.  原文有问题可概括问题也可解决问题（无优选选择）。
*   **科普介绍类：** 客观。选“中性”的概括选项，此时不选对策与问题。
*   **新闻类：** 实时性，新。重视新状况、新成绩、新突破等“新”。（常见结构：背景+新）

### 解题思路
1.  **先看提问：** 分辨题型和文段可能类型。
2.  **再看文段：**
    *   **分析结构：** 在句号处划斜线，再分层，宏观上把握结构，不能“只见树木不见森林”。
        *   宏观上：熟练掌握5种常见结构。
        *   微观上：重视逻辑关联词与指示代词。
    *   **总体思路：**
        *   有重点 $\rightarrow$ 精炼浓缩，同义替换。
        *   无重点 $\rightarrow$ 全面概括，提取共性。
3.  **预设答案：** ① 话题是否一致；② 态度倾向是否一致。
4.  **对比选项：**
    *   选项分析：两出处（六有误、四大类），一比较，一正确，一验证。
    *   **同义替换，对比择优。** 利用选项“挑”答案，允许瑕疵答案出现。
    *   **验证关键信息：** 避免出现低级失误。重视主体、首句、高频词、关键词。

## 02. 结构分析（中心理解题）

### 常见话题引出方式
1.  **开门见山：** 首句确定谈论重点。
2.  **背景知识引出：** 背景不重要。
3.  **对比引出：** 紧跟观点/对策。
4.  **转折引出：** 紧跟观点/对策。
5.  **设问引出：** 注意后续回答。

### 常见对策
1.  **祈使句对策：** 应当、需要等。
2.  **禁止性对策：** 不能、如果……就不……
3.  **必要条件表对策：** 关键、基础、前提、离不开、必不可少等。
4.  **关联词表对策：** 只有……才……、必须……、否则……、要……就……等。
5.  **语义表对策：** 时代命题、重要议题、负有……义务、长远目标、必经之路、必然趋势、有助于等。
*   **如何抓对策？** 多个选项均出对策句，需注意对策句中的主体与必要条件；选项中出现多个不同对策，需注意论证部分论证的内容及对策所针对的问题。

### 五种常见结构分析
1.  **前对策：** 引出话题（问题）$\rightarrow$ 直接给出对策或观点 $\rightarrow$ 论证（重复观点或对策）。
2.  **后对策：** 引出话题（问题）$\rightarrow$ 分析说明 $\rightarrow$ 给出对策或观点。
3.  **总分/分总：**
    *   ① “分”指并列关系，即分论点 $\rightarrow$ 总概括 + 分论点、分论点 + 总结。
    *   ② “分”指总观点后面解释 $\rightarrow$ 总观点 + 分析、分析 + 总结。
    *   **“总”是重点。**
4.  **并列/大并列：**
    *   ① 常见提示：“还、也、另外、同时、另一方面”等，或相同句式引出。
    *   ② 大并列：根据时间、地区、人物等分段、分类分层。
    *   **无重点 $\rightarrow$ 全面概括，提取共性。**
5.  **二层结构：**
    *   可分成前后两部分（A+B）的文段结构，都称之为二层结构。
    *   常见形式：转折、结果/因果、对比（国家对比、时间对比等）、设问+回答……
    *   **并列也是一种考察频率较高的二层结构。**

## 03. 关键信息（中心理解题）

1.  **重视首句：** 很多时候首句确定了整个文段的谈论范围，需重视首句中的主语和关键词。
2.  **重视主体：** 需要注意全文描述的主体，重点关注首句和主旨句的主语。
3.  **重视高频词/关键词：**
    *   **高频词：** 文段中反复、多次出现的词语应重视。
    *   **关键词：** 首句中关键词、观点对策句中的核心词语应重视。

## 04. 选项分析（中心理解题）

### 两出处（排除法）
1.  背景、例子、分析论证中提炼的不选。
2.  “但是/因此/对策”之前、“不是...而是”的“不是...”、被指代部分提炼的选项不选。

### 六有误（常见的错误选项特征）
1.  **片面：** 并列关系只说其一；选项为原文中各个句子中提炼词语拼凑而成。
2.  **杂糅：** 选项为原文中各个句子中提炼词语拼凑而成（逻辑混乱）。
3.  **绝对化：** 最高级、必要条件，原文未提及不选。
4.  **下定义：** 即中性表述，介绍说明，在说理类文段题目中不选。
5.  **无中生有：** 选项中关键信息原文未提及不选。
6.  **本身有误：** 选项本身不符合事实/主流价值观、逻辑错误等不选。

### 四大类（正确选项特征）
1.  **原因类：** 重点谈原因（大篇幅）。
2.  **具体类：** 可归纳概括出具体“小点”（类似申论概括题目）。
3.  **历程类：** 清晰时间轴介绍整个过程。
4.  **区别比较联系类：** 双主语且篇幅相当。

### 解题技巧
*   **一比较：** 选具体的选项。A能涵盖B，那B太具体。
*   **一正确：** 选“A且B”类选项。概括非常全面选项是正确选项概率很高（概率高 $\ne$ 100%）。
*   **一验证：** 在两个选项纠结时，看可否根据选项反推做出文段。

## 05. 标题拟定

**题型说明：** 标题拟定为主旨题的变型，是把主旨意图进行“标题化”的改造。

### 正确选项要求
1.  **标题中需包含关键词（可做适当同义替换）。**
2.  **主语正确。**
3.  **体现作者观点或对策。**
4.  **标题要有可读性、趣味性，对策类标题要有力量。**

**注意：** 标题要特别注意“双引号”选项，是正确选项概率很高。

## 06. 下文推断

**解题思路：** 可将文段和下文看作一个整体，套入常见说理类行文结构。

### 考法分类
*   **三种简单结构：**
    1.  上文文末提出新内容 $\Rightarrow$ 下文解释新内容。
    2.  上文提出问题 $\Rightarrow$ 下文分析问题/分析原因/解决问题。
    3.  上文给出宏观对策 $\Rightarrow$ 下文给出具体对策（成功案例也是对策的一种）。
*   **三种不易识别的结构（盯住文段首句）：**
    1.  对比
    2.  转折
    3.  总分
*   **注意：** 上文已经介绍过的，下文一般不再介绍。

## 07. 语句填入

### 解题思路
1.  **宏观把握：**
    *   填入语句应符合全文行文逻辑（例如对比、转折等）。
    *   衔接句要前后接得当，总结概括句要概括全面。
2.  **细节把握：**
    *   填入语句要和前后句关系正确，要注意语法结构正确（主谓搭配得当）。
    *   逻辑关联词运用得当，可特别考虑句式一致。
    *   全文主语一致，后文指代词指代正确。

### 填入语句在文段中的作用
*   **在开头：** 多为总起或概括。
    1.  前对策（若后文为论证）。
    2.  提出问题（若尾句为对策）。
*   **在中间：** 承上启下。
    *   承和启都是重点。
    *   转折具有倾向性（看倾向前文还是后文，夺还是报）。
*   **在结尾：** 多为总结或对策句。

## 08. 语句排序

### 解题思路
1.  **整体分析：**
    *   套入常见结构：多为对策/后对策、总分、并列/大并列。
    *   用常见结构判断语句顺序，尤其注意转折/对比引出话题。
2.  **盯住细节：**
    *   **看首尾句：** ① 下定义多为首句；② 总结、对策多为尾句。
    *   **找组合：** 通过转折、指代、相同词找到组合。
3.  **借助选项提升做题速度。**

---

## 09. 细节判断

### 解题思路
*   **整体把握：**
    *   细节题没有普适性的解题方法，题与题之间相似性不大，都需要验证每个选项的正误。
    *   一般来说，细节判断题只需要断句，而不需要分析结构。
    *   正确率较低的细节判断题，选项往往存在瑕疵，我们需要抓主要致命问题，而忽略其他。
*   **难点特点：**
    1.  逻辑性极强，“深挖坑”。
    2.  正确选项也存在瑕疵，此时我们需要抓住致命问题，而忽略其他。

### 常见错误选项
1.  **偷换概念/语气/逻辑/数量等。**
2.  **混淆时态。**
3.  **杂糅：** 为原文中各个句子中提炼词语拼凑而成。
4.  **本身有误：** 不符合实际。
5.  **绝对化。**
6.  **无中生有。**
